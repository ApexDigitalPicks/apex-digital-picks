## About This Site

This is a single-page funnel website built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, no build tools required.

**Features:**
- Fully responsive — works on mobile, tablet, and desktop
- Dark navy and blue brand colour scheme
- Sticky navigation with mobile hamburger menu
- Hero section with animated badge and stats
- Product review cards with ratings and commission info
- Email lead capture / newsletter signup
- FAQ accordion
- Scroll-triggered animations
- Zero external dependencies (fonts from Google Fonts only)

## How to Run Locally

No installation needed. Simply open the file in your browser:

```bash
# Clone the repo
git clone https://github.com/yourusername/apex-digital-picks.git

# Open in browser
open index.html
```

Or just double-click `index.html` on your computer.

---

## How to Deploy

### GitHub Pages (free)
1. Push `index.html` to the `main` branch
2. Go to **Settings → Pages**
3. Set branch to `main`, folder to `/root`
4. Save — live in 1–2 minutes

### Netlify (free)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `index.html` file
3. Live instantly

### Tiiny.site (free)
1. Go to [tiiny.site](https://tiiny.site)
2. Upload `index.html`
3. Live in 60 seconds
